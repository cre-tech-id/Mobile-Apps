## Vacuum Cleaning Robot

### About
Vacuum Cleaning Robot serves as a remote control for your cleaning and vacuum robot, allowing you to effortlessly clean dirty floors with the convenience of your smartphone or tablet. Just like operating a car with a remote control, our intuitive app lets you maneuver your cleaning robot across various floor surfaces.

### How to Use
1. First connect the robot to the android manually using Bluetooth.
2. Open the application then turn on Bluetooth by clicking the first leftmost button if Bluetooth is not on.
3. After that, connect the robot to the application by clicking the pair button, then select the robot device.
4. To move the robot press the arrows in the desired direction.
5. If you want to activate cleaning, press the clean button. And if you want to activate the vacuum, click the vacuum button. However, if you want to activate both, click the mix button.
6. Click the auto button if you want to move the robot to automatically clean and vacuum the floor, click the auto button.
